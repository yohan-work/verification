| 항목 | 내용 |
| :--- | :--- |
| **모델** | (예: Claude 3.5 Sonnet) |
| **문제** | (사람도 헷갈리는 복잡한 논리/퀴즈 문제 입력) |
| **일반 프롬프트 결과** | (오답 여부, 풀이 과정의 단순함) |
